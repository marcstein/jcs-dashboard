## Usage Notes

### Form Numbering System
Kentucky's AOC form numbering doesn't follow a strict categorization like Missouri's, but general patterns exist:
- **0XX**: General notices and administrative forms
- **1XX**: Arraignment and pleadings
- **2XX**: Motions, orders, and proceedings
- **3XX**: Warrants, citations, and process
- **4XX**: Victim services and restitution
- **7XX**: Appeals and special writs
- **8XX**: Appeals, certification, and expungement

### Multiple Variants
Many forms have multiple language versions (AR=Arabic, ES=Spanish, FR=French, etc.)

## URL Pattern

All forms follow this URL pattern:
```
https://www.kycourts.gov/Legal-Forms/Legal%20Forms/{FORM_NUMBER}.pdf
```

For language variants, append language code:
```
https://www.kycourts.gov/Legal-Forms/Legal%20Forms/{FORM_NUMBER}_{LANG}.pdf
```

Example: `https://www.kycourts.gov/Legal-Forms/Legal%20Forms/140_ES.pdf` (Spanish Guilty Plea)
